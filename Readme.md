该模板包含用于快速创建 Revit 附加命令的项目模板

## Template Features

- 2020-2024 Revit 版本构建项目
- 支持编译完成自动更新到Revit

## Installation

1. 安装 [.Net SDK](https://dotnet.microsoft.com/download)
2. 运行 `dotnet new install xml.Revit.Templates` 安装项目模板
